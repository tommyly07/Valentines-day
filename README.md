# Valentines-day
hier werde ich für meine Freundin ein Ja oder Nein Frage programmieren, wo sie ankreuzen kann, ob sie mein Valentine sein will!

# Funktionen
Wenn man auf ja drückt dann sollte eine neue Seite kommen wo man sich freut. Und wenn man auf nein drückt, dann sollte mehrere Nachrichten wiederholt werden bis sie ja sagt.
