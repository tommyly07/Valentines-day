# Valentines-day
hier werde ich für meine Freundin ein Ja oder Nein Frage programmieren, wo sie ankreuzen kann, ob sie mein Valentine sein will!

# Funktionen
Wenn man auf ja drückt dann sollte eine neue Seite kommen wo man sich freut. Und wenn man auf nein drückt, dann sollte mehrere Nachrichten wiederholt werden bis sie ja sagt.

# Javascript
Das war jetzt mein erstes Javascript, was eh nicht so sonderlich schwer war. Durch w3schools und AI konnte ich die einfachen Sachen ausführen. Unter Const kann man sich einfach so wie eine Schublade vorstellen, was drinnen ist. Sprich wenn ich const x = 4 habe, dann kann ich nicht später mein x z.B x = 5 einsetzen. Wenn ich die Werte überschreiben will, dann benutze ich let.
